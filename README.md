# 🐟 FishyWebAI

FishyWebAI is an AI-powered phishing website detection system built using Python, Flask, Machine Learning, and Cybersecurity threat intelligence.

## 🚀 Features

* AI-based phishing detection
* Real-time URL analysis
* SSL certificate validation
* Domain age checking
* VirusTotal integration
* Scan history database
* Analytics dashboard
* Threat distribution pie chart
* PDF report generation
* Modern cyberpunk UI

## 🛠️ Tech Stack

* Python
* Flask
* Scikit-Learn
* Pandas
* SQLite
* ReportLab
* HTML
* CSS

## 📊 Dashboard Features

* Total scans
* Safe websites count
* Phishing websites count
* Threat distribution visualization

## 📄 Report Generation

FishyWebAI can generate downloadable PDF security assessment reports for scanned websites.

## ⚙️ Installation

```bash
git clone https://github.com/YOUR_USERNAME/FishyWebAI.git

cd FishyWebAI

pip install -r requirements.txt

python app.py
```

## 🌐 Usage

1. Enter a URL
2. Click Scan
3. View threat analysis
4. Download PDF report
5. View scan history
6. Monitor dashboard analytics
